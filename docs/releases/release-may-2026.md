# Release Notes - May 2026

In May, we focused on improving the onboarding experience for new accounts and adding deeper insights into team operations.

## Major Highlights

### 1. Interactive Setup Checklist
- Overhauled the new user flow by introducing a step-by-step Setup Checklist directly on the Game Day dashboard.
- Guides administrators through sport selection, season configuration, payment tokens, logo uploading, and player invites without leaving the main dashboard.
- Enforced an onboarding gatekeeper to ensure accounts are fully set up before logging game actions.

### 2. Player Availability & Insights
- Integrated pre-match check-ins allowing players to set their availability (Available, Tentative, Unavailable).
- Provides insights on attendance and match roster size.

### 3. Cross-Team Ledger
- Created a consolidated ledger view for Account Admins.
- Allows admins managing multiple teams to view outstanding player balances and billing histories in a single, unified grid.
